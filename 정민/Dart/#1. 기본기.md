### 변수 타입

- **variable**
    
    ```dart
    var name = '변백현';
    name = '백현';
    ```
    
    - 코드는 무조건 위에서부터 아래로 순서대로 실행 된다.
    
    -  똑같은 이름의 변수를 2번 선언할 수 없다.
    

- **integer, double, boolean**
    
    ```dart
    // 정수
    int number1 = 5;
    int number2 = 6;
    
    print(number1 + number2);
    print(number1 - number2);
    print(number1 / number2);
    print(number1 * number2);
    
    // 실수
    double number1 = 5.6;
    
    // 맞다/틀리다
    bool isTrue = true;
    bool isFlase = false;
    
    ```
    

- **글자 타입, String**
    
    ```dart
    String name = '엑소';
    String name2 = '백현';
    
    // 덧셈 가능
    print(name + name2); // 엑소백현
    print(name + ' ' + name2); // 엑소 백현
    
    print('${name} ${name2}'); // 엑소 백현
    print('$name $name2'); // 엑소 백현
    ```
    

- **var vs String**
    - var: type을 오른쪽 값으로 인해서 유추함

- **runtimeType**: 실제로 이 변수의 값이 실행되는 그 순간에 어떤 타입인지를 얄려줌
    
    ```dart
    var name3 = '터래기';
    print(name2.runtimeType); // String
    ```
    

- **dynamic**
    
    ```dart
    dynamic name '백현';
    name = 56;
    ```
    
- **var vs dynamic**
    - var: 한번 선언하면 그 선언할 때 그 타입으로 바로 고정 -> 다른 타입으로 변경 불가능

### null

- **null**: 아무런 값도 있지 않다
- **nullable**: null이 될 수 있다
    
    ```dart
    String? name = '백현';
    name = null;
    ```
    
- **non-nullable**: null이 될 수 없다
    
    ```dart
    String? name2 = '엑소';
    print(name2!);
    ```

### final, const

```dart
final (String) name = '백현';
const (String) name2 = '터래기';
```

- 한번 선언한 값은 변경 불가능
- type 생략 가능
- final: 빌드 타임의 값을 알고 있지 않아도 됨
final DateTime now = DateTime.now();
- const: 빌드 타임의 값을 알아야 함
const DateTime now = DateTime.now(); // 오류
- 빌드 타임의 값을 알아야 한다
= 코드를 작성하는 순간에 그 코드의 값을 알고 있어야 한다.

### Map

```dart
// key, value
Map<String, bool> isHarryPotter = {
	'Harry Potter': true,
	'Ron Weasley': true,
	'Ironman': false,
};

print(isHarryPotter['Ironman']); // false

// 추가
isHarryPotter.addAll({
	'Spiderman': false,
});

isHarryPotter['Hulk'] = false;

// 삭제
isHarryPotter.remove('Harry Potter');

// key값, value값 따로 가져오기
print(isHarryPotter.keys);
print(isHarryPotter.value);
```

### while문

```dart
int total = 0;

while (total < 10) {
	total += 1;
}
print(total); // 10

// do-while
total = 0;

do {
	total += 1;
} while (total < 10);
print(total); // 10

// break
int total = 0;

while (total < 10) {
	total += 1;

	if (total == 5) {
		break;
	}
}
print(total); // 5
```

### 함수

- **매개변수** = parameter = argument
- **positional parameter**: 순서가 중요한 파라미터
- **optional parameter**: 있어도 되고 없어도 되는 파라미터
    
    ```dart
    void main() {
      addNumber(10);
    }
    
    addNumbers(int x, [int y = 20, int z = 30]){
      int sum = x + y + z;
    }
    
    ```
    

- **named parameter**: 이름이 있는 파라미터 (순서가 중요하지 않음)
    
    ```dart
    void main() {
    	addNumbers(y: 20, x: 20);
      addNumbers(x: 10, y: 20, z: 30);
    }
    
    addNumber({
      required int x, 
      required int y, 
      int z = 30,
    }){
      int sum = x + y + z;
    }
    ```
    

- **arrow function** (화살표 함수)
    
    ```dart
    addNumber(int x, {
      required int y, 
    }) => x + y + z;  // 반환되는 값
    ```
